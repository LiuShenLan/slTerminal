## 三、已知注意事项

1. **L4 teardown 噪声**：`Failed to clear mock store` 为 `@wdio/tauri-service` session 销毁时序问题，不影响 spec。
2. **`libpng warning: iCCP`**：WebView2 内嵌渲染管线，非我方代码。
3. **F1 isDisposedRef 复位**：仅在 effect body 首行执行，cleanup 设 true 后、下一次 mount 时 body 复位——**不依赖外部触发**。
4. **F2 RO 闭包安全**：RO 注册在主 effect 内部，依赖 `container` 重建——container 不变的 mount 复用同一 Terminal，RO 持新鲜闭包引用。
5. **F3 windowsBuildNumber 解耦**：主 effect 不再因 IPC 异步返回触发重建。F3 独立 effect 在 Terminal 创建后动态更新 `windowsPty`，不销毁 Terminal。
6. **B1 -NoExit**：`pwsh -NoProfile -NoLogo -NoExit -EncodedCommand <script>`——pwsh 执行完集成脚本后保持交互式 REPL，用户看到提示符。
7. **P1 CSS 高度链**：全局生效——所有 Dockview 面板（终端/编辑器）自动受益，不仅限于终端。
8. **G1b spawn_blocking**：Tokio 阻塞线程池上限 512，slTerminal ~10 终端远未触及。
9. **G2 canFit()**：保留为防御性守卫，fonts.ready 和 RO doFit 仍使用。rAF 轮询路径自行检查 `offsetWidth > 0` 和 `terminal.element`，不依赖 isDisposed 条件。
10. **D3 Compartment.reconfigure()**：不丢失文档状态，编辑器切换语言时保留用户输入内容。
11. **D4 src/ipc/dialog.ts**：薄 re-export，零额外开销，满足架构约束 #1。
12. **E2E 构建前置**：需先执行 `npx tauri build --debug --no-bundle` 再跑 `npm run wdio`，确保 exe 包含最新前端。
