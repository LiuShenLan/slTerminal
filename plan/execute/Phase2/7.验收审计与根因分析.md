## 3. 需求范围变更

用户确认 worktree 功能不需要，需求变更为：
- 删除所有 worktree 相关功能（前端 UI + 后端命令 + 测试）
- 侧栏改为二级树：项目 → 操作页面（去掉 worktree 层级）
- 添加项目不再要求 git 仓库——任意文件夹均可
